Обязательно иметь установленный node js на компьютере

в терминале (cmd/powershell)

Если зависимости удалены (папка node_modules) то пропишите в папках server и client " npm i " без кавычек в терминале

Если зависимости не удалены (папка node_modules) то тогда, Перейти в папку server и ввести в терминал " npm run dev " без кавычек
